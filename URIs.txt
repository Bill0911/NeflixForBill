ACCOUNT:
Create account: 
POST Users/Accounts/register

Activate account:
PUT Users/Accounts/activate

Login
GET Users/Accounts/login

reset Password
PATCH Users/Accounts/reset-password

cancel membership
DELETE Users/Accounts/id


PROFILE:
create profile
POST Users/Accounts/Profiles

update profile
PUT Users/Accounts/Profiles/id

change profile images
PATCH Users/Accounts/Profiles/profile_photo

delete profile
DELETE Users/Accounts/Profiles/id
